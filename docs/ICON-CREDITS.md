# Icon Credits

## Design

The MITM Network Interceptor icon was designed specifically for this project.

## Elements

- **Shield**: Represents protection and security
- **Network nodes**: Represents network traffic
- **Red X**: Represents blocking/interception
- **Lock**: Represents secure proxy
- **Blue color (#007ACC)**: VS Code brand color

## Format

- **Source**: `icon.svg` (vector format)
- **Published**: `icon.png` (256x256 pixels)
- **Style**: Flat design with modern aesthetics
